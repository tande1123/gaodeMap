# gis-amap

> A gis project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 项目结构

```xml
--build     		webpack配置文件
--config    		vue环境相关配置文件
--src
	--assets		开发者资源文件夹
	--components	组件文件夹
	--gis					地图组件文件夹
	--pages			路由页面文件夹
	--router		路由配置文件夹
	--store			vuex配置文件夹
	--filter		过滤器文件夹
	--utils			工具
--static		全局资源文件夹

```
